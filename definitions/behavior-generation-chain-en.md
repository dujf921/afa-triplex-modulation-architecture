# Behavior Generation Chain｜Human Behavior FCA

Observable Output
→ Foreground Entry
→ Historical Template Matching and Explanation
→ New Structure Generation
→ Execution Maintenance
→ Feedback Update

## Observable Output

Observable output is the entry point for analysis, not the conclusion.

## Foreground Entry

Foreground entry determines which inputs receive processing priority.

## Historical Template Matching and Explanation

Historical template matching and explanation maps current input onto stored templates and maintains continuity.

## New Structure Generation

New structure generation produces new causal paths, action structures, and solution spaces.

## Execution Maintenance

Execution maintenance keeps generated paths operating as long-term action and reality-facing output.

## Feedback Update

Feedback update determines whether the next cycle updates, repeats, stabilizes, or locks in.
