# FCA 术语表（标准定义）

| Term | Canonical Meaning | Not This |
| --- | --- | --- |
| SN | 前台进入系统，决定哪些输入进入活跃处理优先级。 | 情绪类型或性格敏感标签。 |
| DMN | 历史模板匹配与解释系统，维持世界、自我、身份、关系与情境连续性。 | 新结构生成系统。 |
| FP | 新结构生成层，负责在旧模板之外生成新因果路径与新行动结构。 | 动机强弱或兴奋度。 |
| ECN | 执行维持系统，把已生成路径维持为持续的现实行动输出。 | 方案发明系统。 |
| Feedback Update | 现实结果回流后更新下一轮行为生成条件的环节。 | 只有“知道结果”但无结构更新。 |
| Hormone Modulation Layer | 调制轴层，调制前台竞争概率、模板权重、生成稳定性与维持稳定性。 | 一一硬对应的独立网络层。 |
| Cortisol | 提高威胁前台化概率并提高 DMN 威胁模板权重。 | 直接生成 FP 新结构。 |
| Dopamine | 提高 FP 占位稳定性与探索推进动量。 | 自动保证产生新结构。 |
| Oxytocin | 降低社会威胁敏感度，并通过连续性/信任背景支持 ECN 维持稳定。 | 爱的度量值或 ECN 本身。 |
