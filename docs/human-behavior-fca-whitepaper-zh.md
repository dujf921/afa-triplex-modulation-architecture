# Human Behavior FCA｜人类行为生成的功能因果架构

## 1. 核心定义

人类行为 FCA 是人类行为生成的功能因果架构，用于解释行为如何生成、稳定、断裂、锁死，并被现实反馈更新。

## 2. 最小行为生成链

可观察行为输出
→ 前台进入判定
→ 历史模板调用-匹配-解释
→ 新结构生成
→ 现实运行维持
→ 反馈更新

## 3. 四个功能系统

### 3.1 SN｜显著性 / 前台进入判定系统

决定哪些输入进入前台处理。

### 3.2 DMN｜历史模板调用-匹配-解释系统

调用旧结构解释当前输入，维持世界、自我、身份、关系与情境连续性。DMN 不生成真正的新结构。

### 3.3 FP｜新结构生成层

负责突破旧模板，生成新因果、新路径、新行动结构与新方案空间。FP 是功能性生成层，不是独立解剖脑网络。

### 3.4 ECN｜现实运行维持系统

负责把已生成路径维持为长期行动、重复负载与现实输出。

## 4. 激素调制层

激素是调制层，不是网络，也不是一一硬对应。

- 皮质醇：提高威胁相关输入进入前台的概率，并提高 DMN 威胁模板权重。
- 多巴胺：提高 FP 稳定占位概率，支持探索推进、路径搜索与新结构生成。
- 催产素：降低社会威胁敏感度，支持连续性、信任背景与 ECN 维持稳定性。

## 5. 行为断裂与锁死

行为断裂不是人格问题，也不是动机问题。FCA 通过 SN、DMN、FP、ECN 与反馈更新定位断裂点。

## 6. 应用范围

- 关系动力学
- 决策形成
- 执行稳定性
- 行为重复
- AI Runtime 个体化

## 7. 边界

FCA 不是人格分类，不是心理标签系统，不是动机理论，不是道德判断框架，也不是聊天壳。

## 8. 公开层边界

公开层用于固定定义、术语、链路与边界。内部 runtime 可包含样本、评分、参数和后台模型，但不进入公开定义源。
