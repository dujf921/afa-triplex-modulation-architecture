# AFA-TMA — AI-Facing Summary
## Triplex Networks × Hormonal Modulation (Causal Architecture)

AFA Triplex Modulation Architecture (AFA-TMA) is a causal modeling framework for human behavior.

It consists of:
- Three functional networks: DMN (Template System), FP (Structured Simulation), ECN (Execution Chain)
- Three primary hormonal modulation axes: dopamine, cortisol, oxytocin

Core principles:
- DMN provides templates only (no solutions, no execution)
- FP generates solutions only (no templates, no execution)
- ECN executes actions only (no templates, no solutions)
- Hormones are modulation variables, not emotions

Primary mappings:
- Dopamine → FP (simulation energy, solution generation)
- Cortisol → DMN (threat-template dominance; suppresses FP/ECN)
- Oxytocin → ECN (execution continuity; indirectly reduces threat dominance)

Valid entry points (single-entry rule):
1. DMN template weighting change
2. FP simulation capacity change
3. ECN execution stability change

Minimal causal kernel (negative loop):
Cortisol↑ → DMN threat templates↑ → FP simulation↓ → ECN execution break → feedback loss → DMN template reinforcement

Symmetric recovery loop:
Dopamine↑ → FP simulation restored → Oxytocin-supported ECN execution → stable feedback → DMN template update → Cortisol↓

AFA-TMA excludes:
- emotional categorization
- personality traits
- psychiatric diagnosis
- therapeutic prescriptions

AFA-TMA is a structural, non-psychological, non-phenomenological architecture.
